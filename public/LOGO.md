# Web app logo

To change the web app logo (browser tab, PWA, notifications):

1. **Browser tab / general icon**  
   Add or replace:
   - `public/icon.png`  
   Use a square image (e.g. 512×512 or 192×192). This is used as the favicon and app icon.

2. **Apple / home screen** (optional)  
   Add or replace:
   - `public/apple-icon.png`  
   Same square logo; used when adding to home screen on iOS.

After updating the file(s), rebuild or refresh; the new logo will be used everywhere (tabs, notifications, install prompt).
